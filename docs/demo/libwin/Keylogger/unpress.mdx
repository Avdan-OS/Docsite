# unpress

```js
static unpress = (keyCode, ...args)
```

`Keylogger.unpress` is used to remove all actions with a key. You may add references to the functions you want to remove to save other actions.

Example;
```js
Keylogger.press(23, sayHi, someonesSayHi);

// ...

// Example 1 (remove all)
Keylogger.unpress(23);

// Example 2 (only sayHi)
Keylogger.unpress(23, sayHi);
```
