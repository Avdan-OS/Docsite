# scrollBarGenerate

```js
scrollBarGenerate = scroll_list_l
```

`scrollBarGenerate` gets a list with initial widget and widget Objects array.

Example from demo:
```js
var scroll_list_g = { 
	"pos" : 0,
	"items" : [
		weather_time,
		player_holder,
	]
}
```

It sets `scroll_list` with the given list to use it in `window.js` independent from the initial variable and origin.

Scrollbar scrolling just changes `pos` and recreates scrollbar.
