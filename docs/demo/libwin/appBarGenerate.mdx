# appBarGenerate

```js
appBarGenerate = apps_list_l
```

`appBarGenerate` looks for an Object with class `.app-bar` and fills it with app Objects. It gets a list of apps with generation configs for each one.

Example from demo:
```js
var apps_list_g = [
	{
		"title" : "Files", // window title
		"src" : "src/images/demo/icons/Apps/Files.png", // appbar/window icon
		"content" : files_content, // content for the window
		"extraClass" : ["file-manager"] // extra classes for window (optional)
	},
    {
        "content" : "hr"
    },
    {
		"title" : "Messages",
		"src" : "src/images/demo/icons/Apps/Messages.png",
		"content" : messages_content,
		"listenerAdder" : messageListeners
	}
}

appBarGenerate(apps_list_g);
```

It sets `apps_list` with the given list to use it in `window.js` independent from the initial variable and origin.
