# About Window

`Window` is used to control window spawning. It controls main id to unite elements with windows and wraps content in windows.

It has 4 methods: [get](get), [inc](inc), [make](make) and [close](close).

`get` is used to get current id and `inc` is used increment id, `make` creates a window Object and `close` is used to close a window.
