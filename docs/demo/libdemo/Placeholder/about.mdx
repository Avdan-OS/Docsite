# About Placeholder

`Placeholder` is used to create defaukt window content. It's easy to use to save place in code adn CSS if there's currently no content for a window.

Example from demo:
```js
var Settings = new Placeholder("Settings", "src/images/demo/icons/Apps/Settings.png");

apps_list = [
    // ...
    {
		"title" : Settings.title,
		"src" : Settings.src,
		"content" : Settings.content
	},
    // ...
]
```
