# Intro to Demo site 

Installing an OS on computer is great step forward with a lot of difficulties. That's why a lot of people want to test an OS to be sure, that it's what they are looking for.

[This demo website](codeheister.github.io/avdan_web/test.html) is created to show the advantages of AvdanOS and to give an opportunity to community to try it's light version even in browser or launch it on PC without any preparations.

That's why demo website should be _lightweight_, _fast_, _multifunctional_ and _fail-safe_.

In this documentation you'll find out, how demo website work and how you can contribute or modify it.
