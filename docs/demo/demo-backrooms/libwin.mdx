# Window library

This library is used to craete windows, store functions and generate dock elements. 

<details>
<summary>Global variables:</summary>

- z_index_g (global `z-index` value to control window visibility)
- win_num_g (global window id iterator)
- apps_list (list of apps for app bar in dock)
- scroll_bar (list of scroll widgets in scroll bar in dock)
- keys (list of currently pressed keys) (great combination with `drag&drop`)

</details>
